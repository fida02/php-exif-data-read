# php-exif-data-read
Using this php code anyone can easily read and show 10 types of exif data of a image

## Avialable exif data
- Camera Brand
- Camera Model
- Lens
- Aperture
- Shutter Speed
- ISO
- Focal Length
- 35mm Focal Length
- Metering Mode
- Metering Mode

## How to use
Copy the full code where you want to show. Then follow the comment in the code.
